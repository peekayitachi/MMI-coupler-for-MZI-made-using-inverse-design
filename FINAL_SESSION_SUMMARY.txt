================================================================================
LESSONS LEARNED
================================================================================

1. DATA DIVERSITY MATTERS
   • 40% synthetic data insufficient for diverse generative model outputs
   • Both MDN and cGAN converged to single mode despite architectural diversity
   • Ensemble sampling more reliable than pure ML when data is limited

2. SOFT CONSTRAINTS BETTER THAN HARD
   • Hard clipping causes gradient saturation
   • Tanh-based soft clipping preserves gradient flow
   • Even soft constraints can't overcome fundamental data limitations

3. PARAMETER BOUNDS ARE CRITICAL
   • Wrong bounds in multiple locations caused cascading failures
   • Always validate scaling matches training data statistics
   • Easy to miss if bounds specified in multiple places

4. FORWARD MODEL CALIBRATION IMPORTANT
   • IL predictions consistently offset by ~2.7dB
   • Indicates possible systematic bias in training data or model
   • Must validate against real physics before using for design decisions

5. ENSEMBLE SAMPLING EFFECTIVE FALLBACK
   • Deterministic sampling guarantees coverage
   • More reliable than pure ML when generative models fail
   • Can combine ML + ensemble for hybrid approach
